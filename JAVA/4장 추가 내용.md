## 기본 생성자 (default constructor)
#### 🔸매개 변수 없고 아무 작업 없이 단순 리턴하는 생성자
#### 🔸디폴트 생성자라고도 부름
## 클래스에 생성자가 하나도 선언되지 않은 경우, 컴파일러에 의해 자동으로 삽입

## 개발자가 클래스에 생성자가 하나라도 작성한 경우
#### 🔸기본 생성자 자동 삽입되지 않음 [ public xxx() {} 기본 생성자가 꼭 있어햐 한다 }
