### 1. 변수의 타입

  ###### ①. 변수의 타입은 저장할 값의 타입에 의해 결정된다.
  
  #### int(정수타입) age = 25(정수);
  #### age(정수타입) = 3.14(실수);
  ####          ↑[불일치]
  
  ###### ②. 저장할 값의 타입과 일치하는 타입으로 변수를 선언
  
  ####   char      ch      =  가;   //char는 문자 타입
  #### [문자형] ↔ 일치 ↔   [문자]

### 2. 값의 타입

 ###### 값 - 문자 - char
 ######    - 숫자 - 정수 - byte, short, int , long
 ######           - 실수 - float, double
 ######    - 논리 - boolean - true or
