# Continue문
### Continue 문은 반복문을 빠져나가지 않으면서 즉시 다음 반복으로 넘어가고자할 때 사용된다. 

#### for(초기문; 조건식; 반복 후 작업){... continue ... ;}  →→→ 반복 후 작업
#### while(조건식){ ... contiune ... ; } →→→ 조건식
#### do{ ... contiune ... } →→→ while(조건식);

<br/>

# break문
### 하나의 반복문을 즉시 벗어난다.

#### for(초기문; 조건식; 반복 후 작업){ ... break; ... } ↓
#### for(초기문; 조건식; 반복 후 작업){ while(조건식) { ... break ; ... } ... ↓ } ...
